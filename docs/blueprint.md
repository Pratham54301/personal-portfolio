# **App Name**: PortfolioFlow

## Core Features:

- React Conversion: Transition existing static HTML portfolio site to React functional components with proper folder structure.
- Routing: Implement React Router for navigation between Home, About, Projects, and Contact pages.
- Animation: Animate page transitions and navbar shrinking on scroll using Framer Motion.
- Preloader: Add a loading animation for a better first impression.
- Responsiveness and Scroll: Use Tailwind CSS and React Scroll or smooth native scroll to implement smooth scrolling and responsiveness.
- Dark Mode Toggle: Toggle between light and dark mode, persisting user preferences locally.
- Contact Form with Nodemailer: Create a contact form with submission to a Node.js backend via Express, utilizing Nodemailer to send form data to prathams54301@gmail.com.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) for a professional and trustworthy feel.
- Background color: Very light gray (#F0F2F5) for a clean and modern look.
- Accent color: Vivid purple (#9C27B0) for CTAs and interactive elements.
- Body text: 'PT Sans', a modern sans-serif, chosen for comfortable readability.
- Headline text: 'Space Grotesk', a versatile sans-serif, providing a modern and professional touch to titles and headings. In longer bodies of text, use 'Inter' instead.
- Use a set of simple, professional icons that complement the modern design.
- Implement subtle, fade-in animations using Framer Motion to create a smooth user experience when sections load, combined with hover effects.
